# Sumo Balls
Este proyecto se ha hecho para el módulo de Diseño Gestión y Producción de Videojuegos del curso de Especialización de Desarrollo de Videojuegos y Realidad Virtual. Consiste en un juego en el que dos bolas se enfrentan una a la otra y el objetivo es echar la bola contraria fuera de la plataforma. Se ha añadido:

#### Mecánica de empujar(Push)
- Ambas bolas pueden empujar al rival si están en contacto. La bola que controla el jugador 1 (bola roja) empuja con la tecla espacio, mientras que la bola que controla el jugador 2 (bola azul) empuja con la tecla control derecha

#### Imagen(Suelo)
- Se ha añadido la carpeta /img con una imagen que representa el suelo sobre el que están las dos bolas
